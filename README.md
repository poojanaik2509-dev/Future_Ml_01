
# Sales Forecasting Dashboard

This project presents a Sales Forecasting Dashboard built using Python and Power BI on the Online Retail Dataset. 
The data was cleaned and prepared using Python, where a forecasting model was developed to predict future sales. 
The processed data was then visualized in Power BI through an interactive dashboard.

Features

* Historical Sales Analysis
* Actual vs Forecast Sales Comparison
* Monthly Sales Trend
* Top 10 Countries by Sales
* Top Selling Products
* KPI Cards (Total Sales, Orders, Customers, Average Order Value, Forecast Sales)
* Interactive Slicers for Data Filtering

 Tools & Technologies

* Python (Pandas, Scikit-learn)
* Power BI
* Online Retail Dataset

This dashboard helps analyze historical sales performance and provides future sales predictions to support business planning and decision-making.
